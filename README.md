# data_profession_survey

## Dashboard Overview
The Power BI dashboard contains interactive elements for a dynamic exploration of the data, with filters for refining the view based on specific roles, skill sets, and experience levels. Each page includes:

Visualizations: Charts, graphs, and infographics make it easy to interpret survey data.
Filters: Interactive options to filter results by specific criteria (e.g., role, industry, skill set).
Insights: Key insights and findings are summarized on each page to guide viewers through the data story.
Getting Started
To view or interact with the Power BI report:

Download the .pbix file from this repository.
Open the file with Power BI Desktop (free to download).
Use the filter options within the report to explore various aspects of the data profession landscape.
Skills Demonstrated
Data Analysis: Processing survey data to uncover actionable insights and trends.
Data Visualization: Designing intuitive and visually appealing charts, graphs, and dashboards in Power BI.
Dashboard Design: Creating a user-friendly and interactive experience to explore key metrics.
